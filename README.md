# Sentiment Analysis using Python

## 📌 Project Overview
This project performs Sentiment Analysis on customer reviews using Natural Language Processing (NLP) techniques. The program classifies reviews as Positive, Negative, or Neutral based on the sentiment of the text.

---

## 🎯 Objectives
- Analyze text data and classify sentiment
- Detect public opinion from reviews
- Visualize sentiment distribution using charts
- Understand sentiment patterns through NLP

---

## ⚙️ Features
- Sentiment classification using TextBlob
- Positive, Negative, and Neutral detection
- Pie chart visualization of sentiments
- Simple and beginner-friendly implementation

---

## 🛠️ Technologies Used
- Python
- Pandas
- TextBlob
- Matplotlib

---

## ▶️ How to Run

Install required libraries:

pip install pandas textblob matplotlib

Run the program:

python sentiment_analysis.py

---

## 📊 Output
The program generates:
- Sentiment classification results in terminal
- Pie chart visualization of Positive, Negative, and Neutral reviews
